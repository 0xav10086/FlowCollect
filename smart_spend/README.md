# 📊 SmartSpend Dashboard - 流量可视化控制台

**SmartSpend** 是 FlowCollect 系统的可视化核心。它通过对接 Go 服务端的 RESTful API，将复杂的流量增量数据转化为极具现代感的动态图表。

---

## ✨ 核心特性

### 1. 响应式流量预警
* **动态视觉响应：** 当日代理流量消耗超过设定阈值时，界面背景通过 `Anime.js` 实现从“安全蓝”到“警告红”的丝滑过渡。
* **泄露高亮：** 若机场扣除流量与本地统计存在显著差异，仪表盘将自动触发闪烁特效提醒审计风险。

### 2. 交互式审计体验
* **节点排行榜：** 基于 `ECharts` 的动态条形图，实时展示各个代理节点的流量权重。
* **数字生命力：** 配合 `Anime.js`，流量数据的增加将以数字滚动的形式呈现，赋予数据实时流动的感官体验。
* **现代化 UI：** 使用 `Tailwind CSS` 与 `Element Plus` 打造，完美适配手机端与 PC 端访问。

## 🛠 技术栈

| 技术 | 说明 |
| --- | --- |
| **Vue 3 (Composition API)** | 核心框架，负责逻辑解耦。 |
| **Vite** | 构建工具，提供极速的热更新体验。 |
| **ECharts** | 数据可视化，呈现流量趋势与节点构成。 |
| **Anime.js** | 动画引擎，负责背景色过渡与数字滚动动效。 |
| **Pinia** | 状态管理，存储服务端获取的实时流量快照。 |

## 🔌 接口集成
前端默认请求后端的 `GET /api/stats` 接口，获取以下格式数据：
* 今日节点消耗排名。
* 代理流量 vs 直连流量比例。
* 设备在线状态汇总。

## 🚀 开发调试

```bash
# 1. 进入目录
cd smart_spend

# 2. 安装依赖
npm install

# 3. 启动开发服务器 (确保 Go 服务端已开启 8686 端口)
npm run dev
```
